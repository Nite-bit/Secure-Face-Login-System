# 🔐 Secure Login System with Face Recognition

A web-based biometric authentication system that uses facial recognition to provide secure login without passwords. The system allows users to register their face, authenticate via webcam, and enables administrators to monitor login activity through a dashboard.

---

## 📌 Project Overview

This project replaces traditional password-based authentication with face recognition using deep learning. It enhances security by reducing identity theft risks and provides a user-friendly interface built with Streamlit.

The system includes multiple modules such as user registration, face authentication, logging, and an admin dashboard for monitoring login attempts.

---

## 🚀 Features

* 👤 Face Registration — Capture and store user face data
* 🔑 Face Login — Authenticate users using facial recognition
* 📊 Admin Dashboard — View login history and analytics
* 📝 Login Logging — Records success and failed attempts
* 🌐 Web Interface — Built with Streamlit for easy interaction
* 🔒 Secure Authentication Workflow
* 📁 Dataset Management
* 🧠 Deep Learning–based Recognition (DeepFace)

---

## 🎯 How It Works

1. Register a user by capturing face images.
2. Login using webcam facial authentication.
3. System compares face with stored dataset.
4. Access granted or denied.
5. Login attempts recorded in logs.
6. Admin dashboard displays analytics.
